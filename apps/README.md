# apps/

This root hosts bounded userspace entrypoints, operator tools, daemons, and demos.

The checked package-role authority is `docs/workspace-package-classification.md`.
This README is stable navigation only, not an app-root inventory table. Read
roles, dispositions, and package counts from the checked authority instead of
this directory note.

App roots are not production-readiness claims. FUSE, ublk, storage-node, scrub,
demo, and CLI behavior must still be validated through issue-scoped checks
before release-facing wording can rely on them.
